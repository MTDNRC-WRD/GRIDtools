# GRIDtools
 Suite of commonly used functions and utilities for working with gridded data.

## Installation
### Current Main Branch
pip install https://github.com/MTDNRC-WRD/GRIDtools/archive/refs/heads/main.zip
### Specific Version
pip install https://https://github.com/MTDNRC-WRD/GRIDtools/archive/refs/tags/*version_number*.zip

Insert the version number you want to download for example v0.2.0
